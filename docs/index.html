About Me

Hi! I am a Third-year Machine Learning and Computer Science student, passionate about building automated solutions.

My fields of interest include Game development, web tool creation, and AI-driven applications.

Currently, I am working at the University of California San Diego's Scripps Institution of Oceanography as a Data Analyst and Web Developer, where I'm responsible for creating workflows and developing web applications to train and deploy large language models for the oceanographic data management team.

My most significant contribution has been developing a submission validator tool that automated the previously manual, hours-long process of validating oceanographic data submissions. The tool performs comprehensive validations in seconds without human error, saving hundreds of hours annually for users and data managers. You can view it here: https://cchdo.github.io/submission-validator/

The tool automatically detects structural issues, such as misaligned columns or incorrect delimiters, and validates parameter names, and name structure, units, and data types against expected standards. It flags missing or inconsistent values in critical fields, ensures measurement flags are applied correctly, and provides clear, actionable feedback to help users quickly resolve errors.
