About Me

Third-year Machine Learning and Computer Science student, passionate about building automated solutions.

My fields of interest include Game development, web tool creation, and AI-driven applications.

Recently I have been working at the University of California San Diego's Scripps Institute of Oceanography as a Data Analyst and Web Developer. My most recent project is our submission validator available here: https://cchdo.github.io/submission-validator/

The tool automatically detects structural issues, such as misaligned columns or incorrect delimiters, and validates parameter names, and name structure, units, and data types against expected standards. It flags missing or inconsistent values in critical fields, ensures measurement flags are applied correctly, and provides clear, actionable feedback to help users quickly resolve errors. Designed with a clean and intuitive interface, the validator simplifies the process of identifying and fixing issues that were previously done by hand and ensures data meets submission guidelines efficiently.
